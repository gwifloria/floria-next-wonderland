# 🌟 Floria's Wonderland

我的个人网站，用 Next.js 搭的。主要用来展示作品、写博客，还有记录一些想法。
考虑到有后端功能，我又有点想省钱，目前线上地址
https://floria-next-wonderland.onrender.com

由于免费，所以可能会碰到 render suspense 了（需要等30s）

## ✨ 功能

- 🎨 **作品集** - 展示项目和技术能力
- 📝 **博客** - 分享技术心得和生活感悟
- 🗺️ **旅行** - 记录去过的地方
- 🎭 **个人故事** - 聊聊经历和想法

## 🛠️ 技术

- **Next.js 14** - 网站框架
- **React 18** - 页面交互
- **TypeScript** - 代码类型
- **Tailwind CSS** - 样式
- **Ant Design** - 组件库
